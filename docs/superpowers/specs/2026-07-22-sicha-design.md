# 시차(視差) — 설계 문서

2026-07-22 · 승인: 사용자 (이름 "시차" 확정, 통합 컨셉 승인, 전월세 API 활용신청 완료)

## 정체성

**시차(視差) — 한 자산을 보는 세 개의 눈.** 같은 아파트에 세 개의 가격이 동시에
존재한다: 시장의 눈(실거래가) · 세입자의 눈(전세가) · 행정의 눈(공시가격).
천문학이 시차(parallax)로 별까지의 거리를 재듯, 세 시선의 각도차를 재면 자산의
리스크가 보인다. 이름의 중의(時差): 공시가는 연 1회, 전세는 2년 주기 — 세 가격은
같은 시각을 살지 않는다. 역전세란 2년의 시차가 만드는 손실이다.

- 3부작 내 위치: 수지(하나의 딜을 해부) · 순환(시장의 시간을 관측) · **시차(한 자산의 가격들을 대조)**
- 타깃 독자: 신탁사·자산운용·시행사 (담보 평가·보증금 리스크·과세 기준의 실무)
- 영문: Parallax · 도메인: sicha.pages.dev (Cloudflare Pages, noindex 기본)
- 위치: `~/시차`

## 미감

순백(#fff 기조) + **적·청 2색** — 애너글리프 입체안경(두 눈의 시차)에서 가져온다.
- 청 = 전세(세입자의 눈) · 적 = 공시(행정의 눈) · 흑 = 실거래(시장의 눈, 기준선)
- 계측 눈금·십자선·가는 괘선. 라이트 기본 + 다크 토글(수지·순환과 동일 체계, ☀/☾ 토글)
- 수지(웜 제도지·주황)·순환(청회·금)과 구분되는 세 번째 미감. 차트는 수지·순환의
  인라인 SVG 엔진(charts.js) 이식 — line/hbars/heatmap/scatter + `--seq-*` 토큰 필수.

## 구성 — 홈 + 5장

| 장 | 제목 | 내용 |
|---|---|---|
| 홈 | 관문 | 세 시선 다이어그램(한 점을 보는 세 개의 눈), 핵심 KPI, 자매 연구 링크 |
| Ⅰ | 세 개의 가격 | 단지 하나(잠실 사례)를 해부 — 세 가격이 어떻게 관측되는가, 데이터 계보 |
| Ⅱ | 세입자의 눈 — 전세가율 | 시군구 지도·분포, 갭(매매−전세), 역전세 관측(8분기 전 대비) |
| Ⅲ | 행정의 눈 — 현실화율 | 공시÷실거래 괴리, 가격대·지역별, 연도별 궤적 |
| Ⅳ | 겹침 — 이중 시차 | 전세가율 × 현실화율 사분면: 보증금 리스크와 담보 착시가 겹치는 지역 |
| Ⅴ | 방법론 | 출처·표본 설계·지표 정의·한계 |

공시가격은 PNU(필지) 단위 조회라 전국 전수가 불가 — **실거래 최다 단지 표본**
(시군구별 거래량 상위 단지의 PNU 역조회: 지오코더→지적→NED 체인, 수지 vworld.py 재사용)으로
구축하고 표본 설계를 Ⅴ장에 명시한다.

## 핵심 지표 정의

- **전세가율** = 동일 단지·전용면적대(±10%)의 전세 실거래 중앙값 ÷ 매매 실거래 중앙값 — 시군구·분기 집계
- **역전세 지표** = 현재 분기 전세 중앙값 ÷ 8분기 전 전세 중앙값 − 1 (음수 = 보증금 반환 압박)
- **현실화율** = 공시가격 ÷ 직전 1년 실거래 중앙값 — 단지·면적 매칭 표본
- 전월세 구분: 보증금만 있는 계약(월세 0)을 전세로 정의. 갱신/신규 구분 필드는 수집하되
  구분 불가 케이스는 한계로 명시.

## 기술 구조 (수지·순환 파이프라인 계승)

```
src/collect/*.py  →  data/*.json     (원천별 수집, config.json 키)
src/analysis/*.py →  out/site_bundle.json  (지표 계산·집계)
src/build/assemble.py → web/index.html    (단일 HTML, 조사-분리·플레이스홀더 검사 포함)
npx wrangler pages deploy web --project-name sicha
```

- 단일 HTML, 외부 네트워크 요청 없음(전 데이터 인라인) — 수지·순환과 동일 원칙

## 한계 (Ⅴ장에 공개할 것)

- 전세가율은 시군구·분기 "중앙값의 비율"이며 개별 계약의 안전도가 아니다
- 공시 표본은 거래 상위 단지 편향 — 저거래 지역 현실화율은 대표성 낮음
- 역전세 지표는 동일 단지 재계약 추적이 아닌 집계 근사
- 상관·괴리 관측이며 인과 주장이 아니다 (순환과 동일한 태도)
